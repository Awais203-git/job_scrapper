# Job Market Analysis Report
## Tools & Techniques for Data Science — Assignment 1

### 1. Data Collection Summary
Job listings were collected from three publicly accessible company career boards across Greenhouse and Lever platforms.

| Company  | Platform   | Jobs Collected |
|----------|------------|----------------|
| Shopify  | Greenhouse | ~150           |
| Stripe   | Greenhouse | ~100           |
| Notion   | Lever      | ~50            |

### 2. Tools Used
- Selenium for browser automation and URL collection
- Scrapy for structured data extraction
- Pandas and Matplotlib for analysis and visualization

### 3. Key Findings
- Most in-demand skills: Python, JavaScript, SQL, TypeScript
- Top hiring company: Shopify
- Majority of roles are remote or hybrid
- Senior level roles dominate the dataset

### 4. Ethical Compliance
- Only publicly accessible pages were scraped
- Polite request delays applied
- ROBOTSTXT_OBEY enabled
- No personal data collected
